## Featured Project

**[🤖 Robotic Manipulator Kinematics Analysis Platform](https://github.com/Meera-Shersheer/Robotic_Manipulator_Kinematics_Analysis_Platform)**

A desktop application for forward and inverse kinematics analysis of industrial robots (UR5, ABB IRB 1600, KUKA KR16). Features symbolic and numeric computation modes, closed-form IK solutions, and an interactive 3D visualizer — built using the Standard Denavit-Hartenberg convention.

`Python` &nbsp;`PyQt6` &nbsp;`SymPy` &nbsp;`Numpy` &nbsp;`OpenGL` &nbsp;`Robotics`
